# Messenger Multi Share

Chrome extension for helping manage screen sharing across multiple Messenger/Facebook group call windows.

## Features

- Detects active Messenger/Facebook group call tabs.
- Starts screen sharing across selected call windows.
- Stops sharing across active call windows.
- Helps refresh Messenger call layout after sharing stops.
- Includes batch camera controls.

## Install locally

1. Open Chrome or Edge extensions page.
2. Enable Developer mode.
3. Click "Load unpacked".
4. Select this extension folder.

## Permissions

This extension uses browser permissions to find call tabs, inject scripts into Messenger/Facebook call pages, control extension UI, and adjust call window sizes during sharing.
